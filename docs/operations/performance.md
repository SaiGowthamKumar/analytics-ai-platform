# Performance

**Purpose:** Establish performance design and measurement principles.

**Owner:** Platform Engineering Lead

Measure end-to-end latency by stage: authorization, semantic retrieval, model generation, SQL validation, query execution, visualization, and insight composition. Set budgets before launch; degrade safely through clarification, cached approved context, asynchronous work, or partial responses. Bound query cost, model tokens, concurrency, and payload size. Profile with representative governed datasets.

## Future expansion

Record SLOs, load profiles, benchmark results, and capacity thresholds.
