# ADR-007: Semantic Knowledge Layer

**Purpose:** Record the governed grounding source for analytics.

**Owner:** Data Stewardship Lead

**Status:** Accepted

## Context

Correct analytics requires governed business meaning beyond raw schemas.

## Decision

Use a versioned, approved semantic knowledge layer as the grounding source.

## Alternatives

Schema-only prompting; per-dashboard definitions.

## Consequences

Greater trust and reuse; stewardship workflow is required.

## Tradeoffs

Curation effort versus ungoverned speed.

## Future expansion

Define asset schemas and approval-service implementation decisions.
