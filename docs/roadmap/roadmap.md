# Roadmap

**Purpose:** Sequence outcomes using milestones rather than dates.

**Owner:** Product Lead

1. **Core Intelligence:** governed natural-language question → semantic context → SQL → results → visualization → insight → recommendation.
2. **Knowledge Layer:** stewardship workflow, versioning, validation, glossary, and semantic coverage.
3. **Enterprise Features:** SSO, RBAC/ABAC, audit, observability, performance, deployment hardening, and multi-tenancy.
4. **Marketplace:** reviewed connector, visualization, and semantic-asset ecosystem.
5. **AI Agents:** bounded multi-step agents, proactive intelligence, and approval-driven actions.

Each milestone exits only with product acceptance, security review, operational readiness, and documentation updates.
