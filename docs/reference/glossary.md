# Glossary

**Purpose:** Maintain shared, unambiguous vocabulary.

**Owner:** Data Stewardship Lead

| Term | Definition |
|---|---|
| Semantic asset | Governed, versioned business-meaning artifact. |
| KPI | Approved metric definition with formula, scope, and owner. |
| Analysis request | A user-initiated, auditable request for analysis. |
| Provenance | Traceable record of sources, versions, policies, and transformations. |
| Port | Inward-facing interface implemented by an adapter. |
| Adapter | Infrastructure or delivery implementation of a port. |
| Data scope | Authorized tenant, source, table, column, and row boundaries. |

## Future expansion

Add business-domain terms through stewardship review.
